# 素材庫

這裡集中存放震欣科技網站專案的原始素材與交接參考檔。

## 資料夾

- `00_目錄索引`：自動產生的素材清單。
- `01_網站原始素材`：各頁面原始文案、照片、產品圖與分類素材。
- `02_型錄簡報與文件`：公司型錄、簡報、PDF、Word 文件。
- `03_交接與參考`：交接說明、臨時參考圖、Logo 原始圖等。

## 注意

目前網站實際部署引用的靜態檔仍位於 `future-site/public/assets`，不要直接搬離該資料夾；若要替換網站上的圖片或 PDF，請先從本素材庫挑選來源，再同步更新 `future-site/public/assets` 內的正式網站檔案與程式引用路徑。
